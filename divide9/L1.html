Классы и объекты

Изучим, как создавать и использовать классы и объекты в C#.

Цель лекции

Понять, что такое классы и объекты, научиться создавать их в языке C#, а также освоить базовые шаги работы с экземплярами классов.

Введение

Объектно-ориентированное программирование (ООП) — это подход, в котором программа строится вокруг объектов. Каждый объект объединяет данные (состояние) и методы (поведение).

Класс — это шаблон или чертёж для создания объектов. Объект — это конкретный экземпляр класса, обладающий своим состоянием и поведением.

Пример из жизни:

  • Класс: «Человек» — содержит общие характеристики (имя, возраст, рост).
  • Объект: «Иван, 25 лет, рост 180 см» — конкретный человек, созданный на основе класса.

Объявление класса

В C# класс объявляется с помощью ключевого слова class:

Пример: простой класс

public class Person
{
    public string Name;
    public int Age;

    public void SayHello()
    {
        Console.WriteLine($"Привет! Меня зовут {Name}, мне {Age} лет.");
    }
}

Здесь класс Person имеет два поля (Name и Age) и один метод SayHello(). Этот класс — как «чертёж», по которому будут создаваться люди.

Создание объекта

Чтобы создать объект из класса, используется ключевое слово new. После этого можно обращаться к его полям и методам:

Пример: создание объекта

class Program
{
    static void Main()
    {
        Person person1 = new Person(); // создание объекта
        person1.Name = "Иван";
        person1.Age = 25;

        person1.SayHello(); // вызов метода
    }
}

Важно: можно создать много объектов одного класса, и каждый будет иметь своё состояние.

Несколько объектов

Пример: два объекта

Person p1 = new Person();
p1.Name = "Мария";
p1.Age = 20;

Person p2 = new Person();
p2.Name = "Алексей";
p2.Age = 30;

p1.SayHello();
p2.SayHello();

Сравнение: класс и объект

Класс Объект
Это описание Это конкретный экземпляр
Содержит поля и методы Хранит значения полей и может вызывать методы
Не занимает память напрямую Занимает память в момент создания через new

Практическое задание

Создайте класс Car с полями Brand и Year, а также методом ShowInfo(), который выводит информацию о машине. Создайте несколько объектов и вызовите метод для каждого.

Подсказка: используйте оператор new для создания объектов и точку . для доступа к полям и методам.
