Методы, параметры и возвращение значений в C#

Изучение основ работы с методами, параметрами и возвращаемыми значениями

Что такое методы в C#

Методы — это блоки кода, которые содержат ряд инструкций. Они позволяют организовать код в логические единицы, которые можно многократно использовать.

Компонент метода Описание Пример
Модификатор доступа Определяет видимость метода public, private, protected
Тип возвращаемого значения Тип данных, который возвращает метод int, string, void (если ничего не возвращает)
Имя метода Уникальный идентификатор метода CalculateSum, PrintMessage
Параметры Данные, передаваемые в метод (int a, int b)
Тело метода Блок кода, выполняющий нужные операции { return a + b; }
C# Пример простого метода 
// Метод без параметров и возвращаемого значения
public void PrintHello()
{
    Console.WriteLine("Hello, World!");
}

// Метод с параметрами и возвращаемым значением
public int CalculateSum(int a, int b)
{
    return a + b;
}

// Вызов методов
PrintHello();
int result = CalculateSum(5, 3);
Console.WriteLine($"Сумма: {result}");

Примечание: В C# все методы должны быть объявлены внутри класса. В отличие от некоторых других языков, в C# нет глобальных функций.

Параметры методов

Параметры позволяют передавать данные в методы для обработки. В C# существует несколько способов передачи параметров.

Тип параметра Описание Синтаксис
Параметры по значению Значение копируется в параметр, исходная переменная не изменяется void Method(int param)
Параметры по ссылке (ref) Передается ссылка на переменную, метод может изменить исходное значение void Method(ref int param)
Выходные параметры (out) Аналогично ref, но переменная должна быть инициализирована в методе void Method(out int param)
Массив параметров (params) Позволяет передавать переменное количество аргументов void Method(params int[] values)
Необязательные параметры Параметры со значениями по умолчанию void Method(int a, int b = 10)
C# Примеры различных типов параметров 
// Параметры по значению (по умолчанию)
void SquareValue(int number)
{
    number = number * number;
    Console.WriteLine($"Внутри метода: {number}");
}

// Параметры по ссылке (ref)
void SquareRef(ref int number)
{
    number = number * number;
    Console.WriteLine($"Внутри метода: {number}");
}

// Выходные параметры (out)
bool TryDivide(int dividend, int divisor, out int result)
{
    if (divisor == 0)
    {
        result = 0;
        return false;
    }
    result = dividend / divisor;
    return true;
}

// Массив параметров (params)
int Sum(params int[] numbers)
{
    int total = 0;
    foreach (int num in numbers)
    {
        total += num;
    }
    return total;
}

// Необязательные параметры
void PrintInfo(string name, int age = 18, string country = "Russia")
{
    Console.WriteLine($"Имя: {name}, Возраст: {age}, Страна: {country}");
}

// Использование
int x = 5;
SquareValue(x); // x останется равным 5
SquareRef(ref x); // x изменится на 25

if (TryDivide(10, 2, out int result))
{
    Console.WriteLine($"Результат: {result}");
}

int total = Sum(1, 2, 3, 4, 5);
PrintInfo("Анна");
PrintInfo("Борис", 25, "USA");

Совет: Используйте выходные параметры (out), когда метод должен вернуть несколько значений. Для сложных сценариев рассмотрите использование кортежей или классов.

Возвращение значений и оператор return

Оператор return используется для возврата значения из метода и завершения его выполнения.

Тип возврата Описание Пример
Возврат значения Метод возвращает конкретное значение указанного типа return 42;
Возврат без значения В методах с void для досрочного завершения return;
Возврат выражения Возвращается результат вычисления выражения return a + b;
Условный возврат Возврат в зависимости от условия if (condition) return true;
C# Примеры использования оператора return 
// Простой возврат значения
int GetAnswer()
{
    return 42;
}

// Возврат результата выражения
double CalculateCircleArea(double radius)
{
    return Math.PI * radius * radius;
}

// Условный возврат
string GetGrade(int score)
{
    if (score >= 90) return "A";
    if (score >= 80) return "B";
    if (score >= 70) return "C";
    if (score >= 60) return "D";
    return "F";
}

// Возврат в методах с void
void ProcessData(string data)
{
    if (string.IsNullOrEmpty(data))
    {
        Console.WriteLine("Данные пустые");
        return; // Досрочное завершение метода
    }
    
    // Обработка данных
    Console.WriteLine($"Обработка: {data}");
    // Неявный return в конце void метода
}

// Возврат нескольких значений с помощью кортежей
(int, int) GetMinMax(int[] numbers)
{
    if (numbers.Length == 0) return (0, 0);
    
    int min = numbers[0];
    int max = numbers[0];
    
    foreach (int num in numbers)
    {
        if (num < min) min = num;
        if (num > max) max = num;
    }
    
    return (min, max);
}

Важно: Для методов с возвращаемым типом (не void) компилятор проверяет, что все возможные пути выполнения содержат оператор return с возвращаемым значением соответствующего типа.

Практические примеры

Рассмотрим несколько практических примеров использования методов в C#.

Рекурсивные методы

C# Пример рекурсивного метода (вычисление факториала) 
long Factorial(int n)
{
    // Базовый случай рекурсии
    if (n <= 1)
    {
        return 1;
    }
    
    // Рекурсивный вызов
    return n * Factorial(n - 1);
}

// Использование
long result = Factorial(5); // 5! = 5 × 4 × 3 × 2 × 1 = 120
Console.WriteLine($"Факториал 5: {result}");

Перегрузка методов

C# Пример перегрузки методов 
public class Calculator
{
    // Версия с двумя целыми числами
    public int Add(int a, int b)
    {
        return a + b;
    }
    
    // Версия с тремя целыми числами
    public int Add(int a, int b, int c)
    {
        return a + b + c;
    }
    
    // Версия с числами с плавающей точкой
    public double Add(double a, double b)
    {
        return a + b;
    }
    
    // Версия с массивом чисел
    public int Add(params int[] numbers)
    {
        int sum = 0;
        foreach (int num in numbers)
        {
            sum += num;
        }
        return sum;
    }
}

// Использование перегруженных методов
Calculator calc = new Calculator();
Console.WriteLine(calc.Add(2, 3)); // 5
Console.WriteLine(calc.Add(2, 3, 5)); // 10
Console.WriteLine(calc.Add(2.5, 3.7)); // 6.2
Console.WriteLine(calc.Add(1, 2, 3, 4, 5)); // 15

Совет: Используйте перегрузку методов для предоставления различных способов вызова метода с разными наборами параметров, но с одинаковой семантикой.
