Элементы управления в WPF

Изучим основные элементы управления WPF: кнопки, текстовые поля, списки и другие элементы интерфейса.

Цель лекции

Научиться использовать основные элементы управления WPF, понять их свойства, события и особенности применения в приложениях.

Основной материал

1. Кнопка (Button)

Элемент Button позволяет пользователю выполнять действие при клике.

Пример XAML
<Button Content="Нажми меня" Width="150" Height="40" Click="Button_Click"/>
Пример C# (Code-behind)
private void Button_Click(object sender, RoutedEventArgs e)
{
    MessageBox.Show("Кнопка нажата!");
}

2. Текстовое поле (TextBox)

TextBox используется для ввода текста пользователем. Позволяет обрабатывать ввод и изменять свойства текста.

Пример XAML
<TextBox Name="textBoxName" Width="200" Height="30" Text="Введите текст" />
Пример C# (Code-behind)
string userInput = textBoxName.Text;
MessageBox.Show($"Вы ввели: {userInput}");

3. Метки (Label)

Label предназначен для отображения текста или информации на форме. Отличается от TextBlock возможностью связывания с элементами управления.

Пример XAML
<Label Content="Введите имя:" />

4. Списки (ListBox и ComboBox)

ListBox и ComboBox позволяют отображать список элементов и выбирать один или несколько из них.

Пример ListBox XAML
<ListBox Width="200" Height="100">
    <ListBoxItem>Элемент 1</ListBoxItem>
    <ListBoxItem>Элемент 2</ListBoxItem>
    <ListBoxItem>Элемент 3</ListBoxItem>
</ListBox>
Пример ComboBox XAML
<ComboBox Width="200">
    <ComboBoxItem>Выбор 1</ComboBoxItem>
    <ComboBoxItem>Выбор 2</ComboBoxItem>
    <ComboBoxItem>Выбор 3</ComboBoxItem>
</ComboBox>

5. Чекбокс (CheckBox) и радиокнопки (RadioButton)

Элементы CheckBox и RadioButton используются для выбора одного или нескольких вариантов.

Пример XAML
<CheckBox Content="Я согласен" IsChecked="False" />

<StackPanel>
    <RadioButton Content="Вариант A" GroupName="Options" />
    <RadioButton Content="Вариант B" GroupName="Options" />
</StackPanel>

6. Панели и контейнеры

Для размещения элементов управления используются панели: Grid, StackPanel, DockPanel, WrapPanel и другие. Панели управляют расположением элементов и их размером.

Пример Grid XAML
<Grid>
    <Grid.RowDefinitions>
        <RowDefinition Height="Auto"/>
        <RowDefinition Height="*"/>
    </Grid.RowDefinitions>

    <Button Content="Верх" Grid.Row="0"/>
    <TextBox Grid.Row="1" Text="Нижняя часть" />
</Grid>

7. События элементов управления

Элементы управления поддерживают множество событий, таких как:

  • Click — нажатие кнопки
  • Checked / Unchecked — изменение состояния CheckBox
  • SelectionChanged — изменение выбранного элемента в ListBox или ComboBox
  • TextChanged — изменение текста в TextBox
Пример обработки события
private void CheckBox_Checked(object sender, RoutedEventArgs e)
{
    MessageBox.Show("CheckBox выбран!");
}

private void ComboBox_SelectionChanged(object sender, SelectionChangedEventArgs e)
{
    ComboBox combo = sender as ComboBox;
    MessageBox.Show($"Вы выбрали: {combo.SelectedItem}");
}

8. Советы по использованию элементов управления

  • Используйте Name для доступа к элементам из C#
  • Для группировки RadioButton используйте атрибут GroupName
  • Используйте панели для гибкого расположения элементов
  • Применяйте стили и ресурсы для единообразного интерфейса
Совет: Для сложных интерфейсов комбинируйте разные панели, чтобы управлять расположением и масштабированием элементов.
