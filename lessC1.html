Типы данных в C#

Изучение простых и структурированных типов данных для эффективного программирования

Простые типы данных (Value Types)

Простые типы данных хранят непосредственно свои значения в стеке. Они эффективны для хранения небольших данных.

Тип данных Размер Диапазон значений Пример
int 4 байта -2,147,483,648 до 2,147,483,647 int age = 25;
long 8 байт -9,223,372,036,854,775,808 до 9,223,372,036,854,775,807 long population = 7800000000L;
float 4 байта ±1.5 × 10⁻⁴⁵ до ±3.4 × 10³⁸ float price = 9.99f;
double 8 байт ±5.0 × 10⁻³²⁴ до ±1.7 × 10³⁰⁸ double pi = 3.1415926535;
decimal 16 байт ±1.0 × 10⁻²⁸ до ±7.9 × 10²⁸ decimal money = 199.99m;
bool 1 байт true или false bool isActive = true;
char 2 байта Символы Unicode char grade = 'A';
byte 1 байт 0 до 255 byte red = 255;
C# Пример использования простых типов 
// Объявление и инициализация переменных
int age = 25;
double height = 1.75;
char initial = 'J';
bool isStudent = true;

// Арифметические операции
int x = 10;
int y = 3;
int sum = x + y;      // 13
int difference = x - y; // 7
int product = x * y;   // 30
int quotient = x / y;  // 3 (целочисленное деление)
int remainder = x % y; // 1 (остаток от деления)

// Преобразование типов
double preciseDivision = (double)x / y; // 3.333...

Примечание: Простые типы эффективны по памяти, но при передаче в методы создается копия значения.

Структурированные типы данных (Reference Types)

Структурированные типы хранят ссылку на данные в куче. Они подходят для хранения сложных структур данных.

Тип данных Описание Пример
string Неизменяемая последовательность символов Unicode string name = "John";
array Коллекция элементов одного типа int[] numbers = {1, 2, 3};
class Пользовательский ссылочный тип class Person { public string Name; }
interface Контракт, определяющий functionality interface IDrawable { void Draw(); }
delegate Тип, представляющий ссылку на метод delegate void MessageHandler(string msg);
C# Пример использования структурированных типов 
// Работа со строками
string firstName = "John";
string lastName = "Doe";
string fullName = firstName + " " + lastName; // Конкатенация
string greeting = $"Hello, {fullName}!"; // Интерполяция

// Работа с массивами
int[] numbers = new int[5] {1, 2, 3, 4, 5};
int firstNumber = numbers[0]; // 1
numbers[1] = 10; // Изменение элемента

// Многомерные массивы
int[,] matrix = new int[2, 3] { {1, 2, 3}, {4, 5, 6} };
int element = matrix[1, 2]; // 6

// Создание и использование класса
public class Person
{
    public string Name;
    public int Age;
    
    public Person(string name, int age)
    {
        this.Name = name;
        this.Age = age;
    }
}

// Использование класса
Person person = new Person("Alice", 30);

Важно: В отличие от простых типов, присваивание ссылочных типов создает копию ссылки, а не данных. Изменения через одну переменную влияют на все переменные, ссылающиеся на те же данные.

Различия между простыми и структурированными типами

Характеристика Простые типы (Value) Структурированные типы (Reference)
Хранение В стеке (Stack) В куче (Heap), ссылка в стеке
Присваивание Копирование значения Копирование ссылки
Сравнение По значению По ссылке (по умолчанию)
Значение по умолчанию 0, false, '\0' null
Производительность Быстрее для небольших данных Медленнее, но гибче
Наследование Не поддерживают Поддерживают
C# Демонстрация различий в поведении 
// Пример с простым типом (копирование значения)
int a = 10;
int b = a; // Создается копия значения
b = 20;
Console.WriteLine(a); // 10 (не изменилось)

// Пример со ссылочным типом (копирование ссылки)
int[] arr1 = new int[] {1, 2, 3};
int[] arr2 = arr1; // Копируется ссылка, а не данные
arr2[0] = 100;
Console.WriteLine(arr1[0]); // 100 (изменилось, так как это те же данные)

Совет: Для сравнения объектов ссылочных типов по значению而不是 по ссылке используйте метод Equals(), который можно переопределить в ваших классах.

Практические примеры и упражнения

Пример 1: Калькулятор площадей

C# Использование различных типов данных 
using System;

class Program
{
    static void Main()
    {
        // Простые типы для расчетов
        double radius = 5.0;
        double circleArea = CalculateCircleArea(radius);
        Console.WriteLine($"Площадь круга с радиусом {radius} = {circleArea:F2}");
        
        // Структурированные типы для хранения данных
        Rectangle rect = new Rectangle(10, 5);
        Console.WriteLine($"Площадь прямоугольника {rect.Width}x{rect.Height} = {rect.CalculateArea()}");
    }
    
    // Метод с простыми типами
    static double CalculateCircleArea(double radius)
    {
        return Math.PI * radius * radius;
    }
}

// Класс как пример структурированного типа
class Rectangle
{
    public double Width { get; set; }
    public double Height { get; set; }
    
    public Rectangle(double width, double height)
    {
        Width = width;
        Height = height;
    }
    
    public double CalculateArea()
    {
        return Width * Height;
    }
}

Упражнение для закрепления

Создайте программу, которая:

  1. Использует простые типы для хранения информации о студенте (имя, возраст, средний балл)
  2. Создает массив студентов (используя класс как структурированный тип)
  3. Реализует методы для вычисления среднего балла всей группы
  4. Выводит информацию о студентах с использованием интерполяции строк

Примечание: Это упражнение поможет закрепить понимание различий между простыми и структурированными типами, а также практику работы с ними.
