Массивы и циклы в C#

Изучение массивов, циклов и их взаимосвязи в языке программирования C#

Массивы в C#

Массивы представляют собой структуры данных для хранения наборов элементов одного типа. Они являются фундаментальной частью большинства языков программирования, включая C#.

Типы массивов

В C# существуют различные типы массивов:

Тип массива Описание Пример объявления
Одномерный Простой массив с одним измерением int[] numbers = new int[5];
Многомерный Массив с несколькими измерениями (например, 2D, 3D) int[,] matrix = new int[3, 3];
Зубчатый (Jagged) Массив массивов, где каждый внутренний массив может иметь разную длину int[][] jagged = new int[3][];
C# Примеры объявления и инициализации массивов 
// Одномерный массив
int[] numbers = new int[5]; // массив из 5 целых чисел
string[] names = { "Анна", "Борис", "Сергей" }; // инициализированный массив

// Многомерный массив (матрица 3x3)
int[,] matrix = new int[3, 3];
matrix[0, 0] = 1; // установка значения элемента

// Зубчатый массив
int[][] jaggedArray = new int[3][];
jaggedArray[0] = new int[2] { 1, 2 };
jaggedArray[1] = new int[3] { 3, 4, 5 };
jaggedArray[2] = new int[1] { 6 };

Примечание: Индексы массивов в C# начинаются с 0. Первый элемент массива имеет индекс 0, второй - индекс 1 и так далее.

Циклы в C#

Циклы позволяют выполнять блок кода многократно, пока выполняется определенное условие. В C# существует несколько типов циклов.

Тип цикла Описание Когда использовать
for Цикл с счетчиком, выполняющийся определенное количество раз Когда известно точное количество итераций
foreach Цикл для перебора элементов коллекции или массива Для перебора всех элементов коллекции без использования индексов
while Цикл, выполняющийся пока условие истинно Когда количество итераций неизвестно заранее
do-while Похож на while, но гарантирует至少 одно выполнение тела цикла Когда нужно выполнить цикл至少 один раз, даже если условие ложно
C# Примеры использования циклов 
// Цикл for
for (int i = 0; i < 5; i++)
{
    Console.WriteLine($"Итерация: {i}");
}

// Цикл foreach
string[] fruits = { "яблоко", "банан", "апельсин" };
foreach (string fruit in fruits)
{
    Console.WriteLine(fruit);
}

// Цикл while
int counter = 0;
while (counter < 3)
{
    Console.WriteLine($"Счетчик: {counter}");
    counter++;
}

// Цикл do-while
int x = 0;
do
{
    Console.WriteLine($"x = {x}");
    x++;
} while (x < 3);

Совет: Используйте цикл foreach для чтения данных из коллекций, когда не нужно изменять элементы или знать их индексы. Для изменения элементов или работы с индексами используйте цикл for.

Взаимосвязь массивов и циклов

Массивы и циклы тесно связаны, так как циклы часто используются для обработки элементов массивов. Без циклов работа с массивами была бы крайне ограничена.

Обработка массивов с помощью циклов

Циклы позволяют выполнять операции над каждым элементом массива:

C# Обработка массива с помощью цикла for 
// Инициализация массива
int[] numbers = new int[10];

// Заполнение массива значениями
for (int i = 0; i < numbers.Length; i++)
{
    numbers[i] = i * 2; // каждый элемент равен удвоенному индексу
}

// Вывод значений массива
for (int i = 0; i < numbers.Length; i++)
{
    Console.WriteLine($"numbers[{i}] = {numbers[i]}");
}

// Поиск суммы всех элементов
int sum = 0;
for (int i = 0; i < numbers.Length; i++)
{
    sum += numbers[i];
}
Console.WriteLine($"Сумма элементов: {sum}");

Использование foreach для перебора массивов

Цикл foreach упрощает перебор элементов массива, когда не нужен доступ к индексам:

C# Использование foreach для работы с массивом 
string[] countries = { "Россия", "США", "Германия", "Китай" };

// Вывод всех элементов массива
foreach (string country in countries)
{
    Console.WriteLine(country);
}

// Поиск элемента в массиве
string search = "Германия";
bool found = false;
foreach (string country in countries)
{
    if (country == search)
    {
        found = true;
        break; // прерывание цикла при нахождении элемента
    }
}
Console.WriteLine($"Страна {search} {(found ? "найдена" : "не найдена")}");

Многомерные массивы и вложенные циклы

Для работы с многомерными массивами часто используются вложенные циклы:

C# Обработка многомерного массива с помощью вложенных циклов 
// Создание и инициализация двумерного массива (матрицы)
int[,] matrix = new int[3, 4];
int value = 1;

// Заполнение матрицы значениями
for (int row = 0; row < matrix.GetLength(0); row++)
{
    for (int col = 0; col < matrix.GetLength(1); col++)
    {
        matrix[row, col] = value++;
    }
}

// Вывод матрицы
Console.WriteLine("Матрица:");
for (int row = 0; row < matrix.GetLength(0); row++)
{
    for (int col = 0; col < matrix.GetLength(1); col++)
    {
        Console.Write($"{matrix[row, col],3}");
    }
    Console.WriteLine();
}

// Вычисление суммы всех элементов матрицы
int total = 0;
foreach (int element in matrix)
{
    total += element;
}
Console.WriteLine($"Сумма всех элементов: {total}");

Примечание: Метод GetLength(dimension) возвращает длину указанного измерения многомерного массива. Для двумерного массива GetLength(0) возвращает количество строк, а GetLength(1) - количество столбцов.

Практические примеры

Рассмотрим несколько практических примеров использования массивов и циклов вместе.

Поиск минимального и максимального значения в массиве

C# Поиск min и max значений в массиве 
int[] numbers = { 12, 45, 67, 23, 9, 56, 78, 34, 90, 11 };

int min = numbers[0]; // предполагаем, что первый элемент - минимальный
int max = numbers[0]; // предполагаем, что первый элемент - максимальный

for (int i = 1; i < numbers.Length; i++)
{
    if (numbers[i] < min)
    {
        min = numbers[i]; // нашли новый минимум
    }
    
    if (numbers[i] > max)
    {
        max = numbers[i]; // нашли новый максимум
    }
}

Console.WriteLine($"Минимальное значение: {min}");
Console.WriteLine($"Максимальное значение: {max}");
Console.WriteLine($"Разность между max и min: {max - min}");

Сортировка массива методом пузырька

C# Сортировка массива методом пузырька 
int[] arr = { 64, 34, 25, 12, 22, 11, 90 };

Console.WriteLine("Исходный массив:");
foreach (int num in arr)
{
    Console.Write(num + " ");
}
Console.WriteLine();

// Реализация алгоритма пузырьковой сортировки
int n = arr.Length;
for (int i = 0; i < n - 1; i++)
{
    for (int j = 0; j < n - i - 1; j++)
    {
        if (arr[j] > arr[j + 1])
        {
            // Обмен элементов
            int temp = arr[j];
            arr[j] = arr[j + 1];
            arr[j + 1] = temp;
        }
    }
}

Console.WriteLine("Отсортированный массив:");
foreach (int num in arr)
{
    Console.Write(num + " ");
}

Работа с зубчатым массивом

C# Пример работы с зубчатым массивом 
// Создание зубчатого массива
int[][] jagged = new int[3][];
jagged[0] = new int[] { 1, 2, 3 };
jagged[1] = new int[] { 4, 5 };
jagged[2] = new int[] { 6, 7, 8, 9 };

// Вывод элементов зубчатого массива
for (int i = 0; i < jagged.Length; i++)
{
    Console.Write($"Строка {i + 1}: ");
    for (int j = 0; j < jagged[i].Length; j++)
    {
        Console.Write(jagged[i][j] + " ");
    }
    Console.WriteLine();
}

// Вычисление суммы всех элементов зубчатого массива
int totalSum = 0;
foreach (int[] innerArray in jagged)
{
    foreach (int value in innerArray)
    {
        totalSum += value;
    }
}
Console.WriteLine($"Общая сумма всех элементов: {totalSum}");

Совет: Для работы с наборами данных в C# также полезно изучить коллекции (List, Dictionary, HashSet), которые предоставляют больше возможностей, чем простые массивы.
